## Cryptocurrency Time Series Forecasting & Analytics Dashboard

## Project Overview
This project focuses on analyzing and forecasting cryptocurrency prices using Time Series techniques. The historical market data was collected using the yfinance API, processed in Python, and visualized through an interactive Power BI dashboard.

The goal of this project is to understand price trends, seasonality patterns, and generate future price forecasts for better decision-making.

---

## Technologies Used
- Python
- Pandas & NumPy
- Matplotlib & Seaborn
- yfinance API
- Time Series Forecasting Models
- Jupyter Notebook
- Power BI
